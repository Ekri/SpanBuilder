Wrapper for spans in Android.

# Usage

**Gradle**
```groovy
buildscript {
  repositories {
    jcenter()
  }

  dependencies {
    compile 'com.maciega.bartosz:span-builder:0.4.1'
  }
 ```



First you should create instance of SpanBuilder:

```java
 SpanBuilder.newInstance("your text here")
 ```
